# AI Threat Detection SOC Project

## Overview

This project simulates a Security Operations Center (SOC) environment where security logs are generated, parsed, and analyzed to detect potential threats using Python.

## Features

* Security log generation
* Attack simulation
* Log parsing and analysis
* Anomaly detection
* Threat visualization dashboard

## Tools & Technologies

* Python
* Pandas
* Streamlit
* Security Log Analysis

## Files

* log_generator.py → generates security logs
* attack_simulator.py → simulates attacks
* log_parser.py → parses log data
* anomaly_detector.py → detects suspicious activity
* dashboard.py → visualizes threats

## How to Run

Install dependencies

pip install -r requirements.txt

Generate logs

python log_generator.py

Run dashboard

streamlit run dashboard.py

## Purpose

This project demonstrates practical SOC analyst skills including log monitoring, anomaly detection, and threat analysis.
